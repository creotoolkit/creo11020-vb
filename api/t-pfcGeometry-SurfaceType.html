Enum EpfcSurfaceType

Description

This enumerated type describes the different Creo Parametric surface types.



Values

EpfcSURFACE_PLANE
A planar surface

EpfcSURFACE_CYLINDER
A cylindircal surface

EpfcSURFACE_CONE
A conic surface

EpfcSURFACE_TORUS
A toroidal surface

EpfcSURFACE_RULED
A surface created by linearly interpolating between two sets of curves.

EpfcSURFACE_REVOLVED
A surface created by revolving a curve about an axis

EpfcSURFACE_TABULATED_CYLINDER
A surface created by linearly projecting a curve

EpfcSURFACE_FILLET
A fillet surface

EpfcSURFACE_COONS_PATCH
A Coons path surface

EpfcSURFACE_SPLINE
A spline surface

EpfcSURFACE_NURBS
A Non-Uniform Rational B-Spline surface

EpfcSURFACE_CYLINDRICAL_SPLINE
A spline surface created using cylindrical coordinates

EpfcSURFACE_SPHERICAL_SPLINE
Reserved for internal use.

EpfcSURFACE_FOREIGN
A foreign surface

EpfcSURFACE_SPL2DER
A spline surface with 2 derivatives

EpfcSurfaceType_nil
Use this enumerated value to represent "null" passed to optional properties or method arguments.
