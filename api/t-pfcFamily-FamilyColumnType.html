Enum EpfcFamilyColumnType

Description

This enumerated type contains the possible types of family table columns.



Values

EpfcFAM_USER_PARAM
A user-defined parameter column, represented by a IpfcFamColParam object.

EpfcFAM_DIMENSION
A dimension column, represented by a IpfcFamColDimension object.

EpfcFAM_IPAR_NOTE
A parameter note column, represented by a IpfcFamColIParNote object.

EpfcFAM_FEATURE
A feature column, represented by a IpfcFamColFeature object.

EpfcFAM_ASMCOMP
A component feature column, represented by a IpfcFamColComp object.

EpfcFAM_UDF
A UDF group column, represented by a IpfcFamColUDF object.

EpfcFAM_ASMCOMP_MODEL
A component model column, represented by a IpfcFamColCompModel object.

EpfcFAM_GTOL
A geometric tolerance column, represented by a IpfcFamColGTol object.

EpfcFAM_TOL_PLUS
A dimension tolerance plus value column, represented by a IpfcFamColTolPlus object.

EpfcFAM_TOL_MINUS
A dimension tolerance minus value column, represented by a IpfcFamColTolMinus object.

EpfcFAM_TOL_PLUSMINUS
A dimension tolerance plus/minus symmetric value column, represented by a IpfcFamColTolPlusMinus object.

EpfcFAM_SYSTEM_PARAM
A system parameter column, represented by a IpfcFamColSystemParam object.

EpfcFAM_EXTERNAL_REFERENCE
An external reference column, represented by a IpfcFamColExternalRef object.

EpfcFAM_MERGE_PART_REF
A merge part reference column, represented by a IpfcFamColMergePart object.

EpfcFAM_MASS_PROPS_USER_PARAM
A mass properties' user-defined parameter column, represented by a IpfcFamColParam object.

EpfcFAM_MASS_PROPS_SOURCE
A mass properties' source column, represented by a IpfcFamColParam object.

EpfcFAM_INH_PART_REF
An inheritance feature part reference column, represented by a IpfcFamColInhFeatPart object.

EpfcFAM_SIM_OBJ
A simulation object column, represented by a IpfcFamColSim object.

EpfcFAM_FEATURE_PARAM
A feature parameter column, represented by a IpfcFamColParam object.

EpfcFAM_EDGE_PARAM
An edge parameter column, represented by a IpfcFamColParam object.

EpfcFAM_SURFACE_PARAM
A surface parameter column, represented by a IpfcFamColParam object.

EpfcFAM_CURVE_PARAM
A curve parameter column, represented by a IpfcFamColParam object.

EpfcFAM_COMP_CURVE_PARAM
A composite curve parameter column, represented by a IpfcFamColParam object.

EpfcFAM_QUILT_PARAM
A quilt parameter column, represented by a IpfcFamColParam object.

EpfcFAM_ANNOT_ELEM_PARAM
An annotation element parameter column, represented by a IpfcFamColParam object.

EpfcFAM_CONNECTION_PARAM
A Mechanism connection column, represented by a IpfcFamColParam object.

EpfcFamilyColumnType_nil
Use this enumerated value to represent "null" passed to optional properties or method arguments.
