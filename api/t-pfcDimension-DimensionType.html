Enum EpfcDimensionType

Description

This enumerated type lists the possible dimension types.



Values

EpfcDIM_LINEAR
A linear dimension.

EpfcDIM_RADIAL
A radial dimension.

EpfcDIM_DIAMETER
A diametrical dimension.

EpfcDIM_ANGULAR
An angular dimension.

EpfcDimensionType_nil
Use this enumerated value to represent "null" passed to optional properties or method arguments.
