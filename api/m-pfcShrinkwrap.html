Module "pfcShrinkwrap"

Description

Contains interfaces for shrinkwrap export.
Enumerated Types
enumEpfcShrinkwrapFacetedFormat
This enumerated type contains the possible formats for the export of a solid model to a faceted shrinkwrap format.
enumEpfcShrinkwrapMethod
This enumerated type contains the possible methods for the export of a solid model to a shrinkwrap format.

Classes
classIpfcShrinkwrapFacetedFormatInstructions
This interface contains instructions for exporting a solid model to a faceted shrinkwrap format.
classIpfcShrinkwrapFacetedPartInstructions
This interface contains instructions to export a solid model to a shrinkwrap faceted part.
classIpfcShrinkwrapMergedSolidInstructions
This interface contains instructions to output a solid model to a shrinkwrap model while merging components.
classIpfcShrinkwrapModelExportInstructions
This interface contains the instructions used for export of a solid model to a shrinkwrap format.
classIpfcShrinkwrapSTLInstructions
This interface contains instructions to export a shrinkwrap STL file.
classIpfcShrinkwrapSurfaceSubsetInstructions
This interface contains instructions to export a shrinkwrap model using a subset of surfaces.
classIpfcShrinkwrapVRMLInstructions
This interface contains instructions to export a model to a shrinkwrap VRML file.
